29    /**
30    * Instant is the standard implementation of a fully immutable instant in time.
31    * <p>
32    * <code>Instant</code> is an implementation of {@link ReadableInstant}.
33    * As with all instants, it represents an exact point on the time-line,
34    * but limited to the precision of milliseconds. An <code>Instant</code>
35    * should be used to represent a point in time irrespective of any other
36    * factor, such as chronology or time zone.
37    * <p>
38    * Internally, the class holds one piece of data, the instant as milliseconds
39    * from the Java epoch of 1970-01-01T00:00:00Z.
40    * <p>
41    * For example, an Instant can be used to compare two <code>DateTime</code>
42    * objects irrespective of chronology or time zone.
43    * <pre>
44    * boolean sameInstant = dt1.toInstant().equals(dt2.toInstant());
45    * </pre>
46    * Note that the following code will also perform the same check:
47    * <pre>
48    * boolean sameInstant = dt1.isEqual(dt2);
49    * </pre>
50    * <p>
51    * Instant is thread-safe and immutable.
52    *
53    * @author Stephen Colebourne
54    * @since 1.0
